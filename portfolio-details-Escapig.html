  • Skills: C# Unity Gameplay/3C UI Sound Integration Project Management

Pitch

Help Dandinou, El Piggo, Justine, and Petrick escape the pig slaughterhouse and save their friends in this brand-new puzzle game. You will have to use the abilities of each pig and make them cooperate in order to solve the different puzzles of the game.

This game was created during the 2023 Game Week at Isart Digital, with the theme "One for all and all for one."


My Tasks

  • Creation of the 4 different pigs with their movements, capacities, and animation implementation
  • Created the switch system between different pigs
  • Implemented level design elements (Pipes and movable boxes)
  • Sound integration using FMod
  • Menus/UI integration
